# AWS EC2 Reporting with Boto3

Python scripts for provisioning AWS EC2 instances and generating detailed CSV reports of instance information using the Boto3 SDK.

## Project Overview

This project demonstrates foundational AWS automation using Python and Boto3, focusing on EC2 instance management and programmatic report generation.

## What It Does

- Provisions EC2 instances programmatically
- Retrieves instance details including IDs, IP addresses, instance types, and tags
- Exports instance data to CSV reports
- Demonstrates core Boto3 patterns for AWS automation

## Technologies Used

- Python 3
- AWS EC2
- Boto3 SDK
- AWS CLI

## Prerequisites

- Python 3 installed
- AWS CLI configured with valid credentials
- Required packages: `pip install -r requirements.txt`
